# DompetKampus 💰
Aplikasi Pencatat Keuangan Sederhana (Expense Tracker)

Dibuat menggunakan React Native + Expo

## Fitur
- Menampilkan total saldo secara otomatis
- Input transaksi Pemasukan & Pengeluaran
- Riwayat transaksi dengan warna hijau/merah
- Dibangun dengan State Array dan FlatList
